# Divider

分割线

| 属性          | 说明                            | 类型     | 默认值       | 可选值                |
| ------------- | ------------------------------- | -------- | ------------ | --------------------- |
| type          | 分割线的类型                    | `string` | `horizontal` | `horizontal/vertical` |
| dashed        | 是否为虚线                      | `bool`   | `false`      | `true/false`          |
| color         | 分割线的颜色，当没有title时有效 | `string` | ` `          |                       |
| titlePosition | title位置                       | `string` | `center`     | `center/left/right`   |
